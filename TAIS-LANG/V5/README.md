# TAIS-LANG v5 — Understanding · Syntax · Culture

No LLM. No pretrained language model. No codebook sentence generation.

v5 expands the ecological speech world with a hard audit layer: speech is only
counted as communication when a listener interprets it, acts, and the outcome
is measured.

**New in v5:**
- Understanding audit: every listener action is logged with intent, interpretation, outcome
- Semantic rate vs utility rate: distinguishes useful action from true understanding
- Large structured world (32x32) with food, water, shelter, poison, landmarks
- Multi-need survival: energy, hydration, toxicity, predation
- Headless training with save/load colony support

```bash
pip install flask flask-cors
python swarm_v5.py
# in another terminal:
npm install && npm run dev
```
