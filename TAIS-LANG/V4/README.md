# TAIS-LANG v4 — World · Memory · Reference

Motes remember events, inherit memories, leave stone markers, and learn from the world.

- Memory buffer: each mote remembers recent events, speech, and teaching
- Inherited memories: newborn motes get a partial copy of parent's memory
- Stone markers: motes leave stones containing memories for others to read
- Stone lexicon: motes can read stones and learn grounded tokens

pip install flask flask-cors
